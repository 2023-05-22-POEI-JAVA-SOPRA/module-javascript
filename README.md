# TP d'évaluation de module

Développez une application de gestion de contacts en JavaScript afin de permettre aux utilisateurs de gérer leur liste de contacts. Voici les fonctionnalités requises de l'application :

- Création d'un contact
- Affichage de tous les contacts
- Affichage d'un contact individuel
- Suppression d'un contact

**Exigences :**

- Les contacts créés doivent être enregistrés dans le stockage local (local storage) afin de partager les données entre les différentes pages.
- Les données à stocker sont les suivantes : nom, prénom, email, téléphone et poste. Il est important de noter que tous les champs sont obligatoires.
- Pour l'affichage d'un contact individuel, il est nécessaire de présenter les données dans une nouvelle page.
